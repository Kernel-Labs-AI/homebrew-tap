# Homebrew Tap for AWT

This is the official Homebrew tap for [AWT (Agent WorkTrees)](https://github.com/kernel-labs-ai/awt).

## Installation

```bash
brew install kernel-labs-ai/tap/awt
```

## What is AWT?

AWT (Agent WorkTrees) is a CLI tool that enables multiple AI agents to safely create, use, and hand off Git worktrees for concurrent development workflows.

## Available Formulas

- **awt** - Agent WorkTrees CLI tool
